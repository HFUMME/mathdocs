
# Mathdocs

A simple HTML-only template for teaching material containing math equations.

## Get repository

1. Sign Up on https://github.com
2. Install the git version control software: https://git-scm.com/downloads
3. Install the github desktop software: https://desktop.github.com
4. Authenticate in github desktop: https://docs.github.com/en/desktop/installing-and-configuring-github-desktop/authenticating-to-github
5. Clone this repository (see green button above)
   * Alternative: `git clone git@github.com:HFUMME/mathdocs.git`

## Install Editor

1. Install Visual Studio Code: https://code.visualstudio.com
2. In Visual Studio Code install the Live Server extension
    * Search for "Live Server" in the extensions panel (the symbol with the Tetris-like shape)
3. In Visual Studio open the folder where the git repository has been stored
4. Switch to the files panel (the symbol with the two sheets of paper) and right-click on "index.html" and choose "Open with Live Server"
